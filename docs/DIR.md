## 📝 문서 작성 가이드

새로운 문서를 추가할 때는 다음 규칙을 따르세요:

1. **파일명**: 대문자로 시작, 하이픈(-)으로 단어 구분 (예: `MY_GUIDE.md`)
2. **형식**: Markdown 사용
3. **구조**: 목차, 섹션, 코드 예제 포함
4. **언어**: 한국어로 작성 (비개발자 친화적)
